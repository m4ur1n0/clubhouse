## ClubHouse
https://clubhouse-bb0e602288cc.herokuapp.com/
An app to allow users to find and create clubs.

## Members
Alex Shen, Theo Maurino, Neo Trovela-Villamiel

### MVP
 - A front-end website where clubs post information such as meeting time, location, description, and a link to a chat group / space.
 - The messaging app will allow users to join and switch between different spaces (and possibly dm members) (MVP - Slack)

### User Stories
User story - club founder:
- Alice wants to create a cryptography club. She can create a club page and add all necessary information such as description, meeting time, and meeting place. Doing so also creates a chat room for other members.

User story - Freshman:
- Bob is a freshman and want to join Alice's cryptography club. He finds her club page on ClubHouse and really likes the posted description. He then joins the club space via a link and can communicate with club members.
